# LeetCode Solutions

A growing collection of my C++ answers to LeetCode problems. Each solution lives in its own file and follows LeetCode's `Solution`-class format, so it can be copied straight into the problem editor.

The focus here is practice: working through common data-structures-and-algorithms patterns, comparing approaches, and keeping a record of solutions as I learn. The code is intentionally kept close to the submitted version rather than packaged as a standalone application.

## Repository layout

```text
LeetCode/
├── Q1.cpp       # problem 1
├── Q4.cpp       # problem 4
├── ...
├── Q1903.cpp    # problem 1903
└── README.md
```

Files use the naming convention `Q<problem-number>.cpp`.

## Using a solution

1. Open the problem link from the table.
2. Open the matching `Q<number>.cpp` file.
3. Paste its `Solution` class into LeetCode's editor and run it there.

LeetCode supplies the surrounding headers, standard-library types, and test harness. For that reason, these files are not all intended to compile by themselves without adding the necessary includes and a `main` function.
