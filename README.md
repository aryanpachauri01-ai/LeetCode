## Structure

Each file is named `Q<problem-number>.cpp` and contains a single `Solution` class matching LeetCode's submission format, so any file can be pasted directly into the LeetCode editor.

```
LeetCode/
├── Q1.cpp
├── Q4.cpp
├── Q7.cpp
├── Q9.cpp
├── Q12.cpp
└── README.md
```

## Usage

```bash
git clone https://github.com/aryanpachauri01-ai/LeetCode.git
cd LeetCode
```

Open any `Q<number>.cpp` file to view the solution for that problem.
